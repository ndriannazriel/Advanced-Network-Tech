## Configure LLDP on switches

##### PHYSICAL SWITCH
```
conf t
lldp run
exit

int 
lldp transmit
lldp receive
```

##### Verify
```
show lldp neighbors detail
```

You should be able to see the other RGW physical routers(Other members) when this command is used.
